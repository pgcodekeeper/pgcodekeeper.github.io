Помощь

Возможности продукта

Дерево процессов с отображением блокирующих и ожидающих запросов

Сортируемое дерево процессов позволяет легко ориентироваться среди всех процессов, находить заблокированные ресурсы, блокирующие и ожидающие запросы.

Отмена или уничтожение процесса

Послать сигнал отмены или уничтожения процесса в один клик.

Всегда актуальная информация

Автообновление дерева процессов по действию пользователя (отмена/уничтожение процессов), по настраиваемому таймеру или вручную.

Информирование пользователя, даже при свернутом окне приложения

Нотификации и изменение иконки в трее сигнализирует о появлении в одной из БД заблокированных процессов.

Сохранение и загрузка истории блокировок

Сохранение в файл и загрузка обратно в приложение истории блокировок - для быстрого решения проблем с последующим анализом сложившейся ситуации.

Как установить

Для установки и запуска pgSqlBlocks выполните следующие шаги:

1
Установите Java JRE версии 1.8 и выше для вашей платформы. Скачать можно по ссылке.
2
Скачайте последнюю версию pgSqlBlocks для вашей платформы по ссылке.
3
Запустите pgSqlBlocks двойным кликом мыши по скачанному jar-файлу либо через консоль командой java -jar pgSqlBlocks-1.3.7-Linux-64.jar
