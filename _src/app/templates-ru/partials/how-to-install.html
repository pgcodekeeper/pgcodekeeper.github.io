pgCodeKeeper является дополнением к среде разработки Eclipse. Для установки pgCodeKeeper выполните следующие шаги:

  1. Установите Java SDK для вашей платформы (требуется для запуска Eclipse).
  2. Установите платформу Eclipse.
  3. Запустите Eclipse. Перетащите кнопку Install на главное окно Eclipse Или выберите пункт меню «Help» - «Install new software», в открывшемся диалоге в поле «Work with» введите URL http://pgcodekeeper.org/update/ и нажмите «Enter». В списке доступных плагинов выберите pgCodeKeeper и проведите инсталляцию плагина согласно подсказкам мастера установки.
  4. Перезагрузите Eclipse. Теперь pgCodeKeeper должен быть успешно установлен!

Версию для неинтерактивной работы pgCodeKeeper можно взять на Github.
