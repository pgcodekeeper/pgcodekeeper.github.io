pgCodeKeeper является дополнением к среде разработки Eclipse. Для установки pgCodeKeeper выполните следующие шаги:

  1. Установите Java SDK для вашей платформы (требуется для запуска Eclipse). Скачать можно по ссылке.
  2. Установите платформу Eclipse.
  3. Запустите Eclipse. Выберите пункт меню «Help» - «Install new software», в открывшемся диалоге в поле «Work with» введите URL http://pgcodekeeper.ru/update/release/ и нажмите «Enter». В списке доступных плагинов выберите pgCodeKeeper и проведите инсталляцию плагина согласно подсказкам визарда установки.
  4. Перезагрузите Eclipse. Теперь pgCodeKeeper должен быть успешно установлен!

Версию для неинтерактивной работы pgCodeKeeper можно взять на Github.
