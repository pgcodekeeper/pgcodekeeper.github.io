pgCodeKeeper is an Eclipse IDE plug-in. Please follow these steps to install pgCodeKeeper:

  1. Install Java SDK for your platform (required to run Eclipse). Click this link to download.
  2. Install Eclipse platform.
  3. Launch Eclipse. Use menu option «Help» - «Install new software», enter http://pgcodekeeper.org/update/ URL in the dialogue's «Work with» field and press «Enter». Choose pgCodeKeeper from the list of available plug-ins and install the plug-in according to the installation wizard prompts.
  4. Restart Eclipse. pgCodeKeeper has been successfully installed.
