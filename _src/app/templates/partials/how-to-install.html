pgCodeKeeper is an addition to the Eclipse development framework. Please do the following steps to install pgCodeKeeper:

  1. Install Java SDK for your platform (required for the Eclipse starting). Click this link to download.
  2. Install the Eclipse platform.
  3. Start the Eclipse. Choose the menu option «Help» - «Install new software», enter URL http://pgcodekeeper.org/update/ in the opened dialogue in the field «Work with» press «Enter». Choose pgCodeKeeper from the list of available plugins and install the plugin according to the installation wizard prompts.
  4. Reload the Eclipse. Now pgCodeKeeper has been successfully installed.

pgCodeKeeper CLI version is available on Github.
