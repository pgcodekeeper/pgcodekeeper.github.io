How to install

First, you should install Java SDK for your platform (required to run Eclipse) and install Eclipse.

Eclipse Marketplace

  1. The easiest way to install pgCodeKeeper is Eclipse Marketplace: launch Eclipse, version 4.6 or higher, with the installed Eclipse Marketplace, and select Help > Eclipse Markeplace….
  2. Enter “pgCodeKeeper” into the search box.
  3. Select the pgCodeKeeper package, review the Terms of service, and click Finish for the installation of the latest pgCodeKeeper version to begin.
  4. After the installation is finished, restart Eclipse to apply the changes.
  5. Launch Eclipse and select: Window > Perspective > Open Perspective > Other > pgCodeKeeper.

Update site

  1. Another way to install pgCodeKeeper is the update website: launch Eclipse, version 4.6 or higher, select Help > Install New Software… Plug-in installation (with the help of the update web-site) wizard will appear on the screen.
  2. In the Work With field specify the path to the update website: http://pgcodekeeper.org/update/.
  3. After pressing Enter you’ll see the available plug-ins in the list below.
  4. Select the pgCodeKeeper package, review the Terms of service, and click Finish for the installation of the latest pgCodeKeeper version to begin.
  5. After the installation is finished, restart Eclipse to apply the changes.
  6. Launch Eclipse and select Window > Perspective > Open Perspective > Other > pgCodeKeeper.

Separate pgCodeKeeper build

You can install pgCodeKeeper with a separate build. You can download pgCodeKeeper build here. Apart from regular installation, you can install pgCodeKeeper in your Eclipse IDE.

CLI version

pgCodeKeeper CLI version is available on Github.
